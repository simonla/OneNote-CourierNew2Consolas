# OneNote-CourierNew2Consolas
OneNote代码宏，转换Courier New字体到Consolas，获取的范围为当前页面

我经常会使用OneNote做一点笔记，自然也用到了某个代码高亮的插件，但是，等宽字体fallback到的`Courier New`我觉得不是很好看，于是有了这个脚本
